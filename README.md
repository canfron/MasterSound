# MasterSound

API REST para gestión de ventas, inventario, pedidos y facturación.

## Endpoints

- `/` - Saludo
- `/sales` - Ventas
- `/inventory` - Inventario
- `/orders` - Pedidos
- `/prices` - Precios
- `/customers` - Clientes (CRUD)
- `/invoices` - Facturación

## Stack

- Python 3.11+
- FastAPI
- Pydantic
- SQLite (próximamente)
